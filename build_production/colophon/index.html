This website is build with Jigsaw.

Bootstrap 3 was used to build the front-end.

Icons on the website are Font Awesome icons.

Fonts used on the website are Lato, Noto Serif and Source Code Pro.

Most colors used throughout the website are based on the Flat UI Colors page.

The homepage image was taken by Stefan Neubig.

The homepage image uses the Backstretch plugin by Scott Robin.

Favicons were based on the Favicon cheat sheet by Audrey Roy.

Code blocks are highlighted by the Highlight.js plugin. The theme that's being used is Solarized Dark.

Comments are powered by Disqus.

The website is hosted on Github Pages.

The header images are from Unsplash.
